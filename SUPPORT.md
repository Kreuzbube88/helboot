# Support

## Where to get help

- **Documentation:** Start with the [README](README.md) and the
  [docs/](docs/) directory.
- **Questions & ideas:** Open a
  [GitHub Discussion](https://github.com/Kreuzbube88/helboot/discussions)
  if enabled, otherwise open an issue with the `question` label.
- **Bugs:** Open a [GitHub issue](https://github.com/Kreuzbube88/helboot/issues)
  and include: HELBOOT version, deployment method (Docker/Unraid), network
  mode (Proxy DHCP or HELBOOT DHCP), and relevant log output from the UI
  log viewer.
- **Security issues:** Never report these publicly — see
  [SECURITY.md](SECURITY.md).

## Scope

HELBOOT targets homelab environments. Best-effort support is provided by
maintainers and the community; there is no commercial support offering.
